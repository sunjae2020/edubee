convert-to-contract logic (wrap in db.transaction):
  1. INSERT contracts { name: 'CT-{accountName}-{YYYY-MM-DD}' }
  2. Copy quote_products → contract_products (preserve serviceModuleType)
  3. For each unique serviceModuleType, auto-create _mgt record:
       study_abroad  → INSERT study_abroad_mgt  { contractId }
       pickup        → INSERT pickup_mgt         { contractId }
       accommodation → INSERT accommodation_mgt  { contractId }
       internship    → INSERT internship_mgt     { contractId }
       settlement    → INSERT settlement_mgt     { contractId }
       guardian      → INSERT guardian_mgt       { contractId }
  4. UPDATE contracts SET service_modules_activated = [activated list]
  5. UPDATE quotes SET quote_status = 'Accepted'
  Return { contractId, activatedModules: string[] }
