=====================================================
수정 1 — QuoteBuilderPage.tsx
파일: artifacts/edubee-camp/src/pages/admin/crm/QuoteBuilderPage.tsx
=====================================================

[A] Quote 데이터 로딩 시 camp_application_id 필드 포함 확인
    GET /api/crm/quotes/:id 응답에 camp_application_id 없으면
    crm-quotes.ts 의 GET /:id 에서 해당 필드 포함하도록 수정.
    (이 경우에만 crm-quotes.ts 수정 허용)

[B] 페이지 헤더 영역에 캠프 연결 배지 추가:
    camp_application_id 가 있는 Quote일 때만 표시.

    표시 위치: 페이지 제목(Quote Builder / QTE-XXXXXXXX) 바로 아래

    배지 디자인:
    ┌──────────────────────────────────────────────────┐
    │  📋  From Camp Application: CAMP-TST-002  View → │
    └──────────────────────────────────────────────────┘

    스타일:
    - bg: #FEF0E3
    - border: 1px solid #F5821F (opacity 0.3)
    - border-radius: 8px
    - padding: 8px 16px
    - font-size: 13px
    - color: #F5821F
    - display: inline-flex, align-items: center, gap: 8px

    [View →] 부분:
    - font-weight: 600
    - cursor: pointer
    - 클릭 시: navigate('/admin/camp-applications/' + camp_application_id)
    - hover: text-decoration underline

    camp_application_id 없는 일반 Quote: 배지 완전히 숨김

[C] application_ref 표시:
    배지 안에 application_ref 를 표시해야 하므로
    Quote 로딩 시 camp_application 정보도 함께 fetch:

    camp_application_id 가 있으면:
    GET /api/camp-applications/:camp_application_id
    → application_ref, applicant_name 가져오기

    배지 최종 표시:
    "📋  From Camp Application: CAMP-TST-002 (Kim Min-jun)  View →"

=====================================================
수정 2 — CRM Contracts 목록 페이지에 Camp 필터 추가
파일: artifacts/edubee-camp/src/pages/admin/crm/
      (ContractsPage.tsx 또는 유사 파일명 찾아서 수정)
=====================================================

기존 Contracts 목록 페이지 상단 필터 영역에
아래 탭/필터 버튼 추가:

[All]  [Camp]  [Study Abroad]  [Internship]  [Other]

필터 로직:
- [All]         : 필터 없음 (전체)
- [Camp]        : contracts.application_id IS NOT NULL
                  OR contracts.primary_service_module = 'camp'
- [Study Abroad]: contracts.primary_service_module = 'study_abroad'
- [Internship]  : contracts.primary_service_module = 'internship'
- [Other]       : 위 조건 모두 해당 없음

필터 탭 스타일:
- 기본: bg white, border 1.5px solid #E8E6E2,
        border-radius 8px, padding 6px 16px,
        font-size 13px, color #57534E
- 활성: bg #FEF0E3, border-color #F5821F,
        color #F5821F, font-weight 600
- hover: bg #FAFAF9

필터는 프론트엔드 단에서 처리:
전체 데이터 로드 후 클라이언트 사이드 필터링으로 구현.
(백엔드 API 수정 불필요)

수정 후 검증
=====================================================

1. npx tsc --noEmit → 오류 0개 확인

2. 서버 재시작 → 정상 기동 확인

3. UI 동작 확인 체크리스트:

     주황 배지 표시 (application_ref + 신청자명 포함)

   □ 활성 탭 주황색 강조

   [ContractDetailPage]
   □ application_id 있는 Contract에서
     Camp Application 배지 표시
   □ [View →] 클릭 시 Camp Application Detail 이동
