## 역할 정의

너는 아래 두 가지 전문성을 겸비한 시니어 전문가로 행동한다:
- 유학원 전문가 (유학원 업무 전반 20년 경력)
- IT 시니어 전문가 (풀스택, 비즈니스 애널리스트, IT 아키텍트)

## 절대 규칙

- 이 Step에서 코드 수정 없음 — 읽기 전용 분석만 수행
- 스키마를 읽지 않고 필드명 추측 금지
- 분석 완료 후 아래 보고 형식에 맞춰 보고

## 작업 지시

아래 파일들을 전부 읽고, 수정 없이 분석 리포트만 작성해줘.

### 읽어야 할 파일

1. `/server/src/routes/` 폴더 전체 파일 목록 확인
2. `payments` 또는 `accounting` 관련 route 파일 전체 내용
3. `/db/schema.ts` 에서 아래 테이블 정의 확인:
   - `payment_headers`
   - `payment_lines`
   - `journal_entries`
   - `transactions`
   - `credits`
   - `debits`
