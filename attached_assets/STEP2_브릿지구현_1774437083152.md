## 추가할 브릿지 로직 (정확히 이대로 구현)

### 브릿지 시작 주석 (필수)
```typescript
// === BRIDGE: auto-create transaction record ===
```

### 1단계 — transactions 레코드 자동 생성

```typescript
const transactionName = `TRANSACTION-${new Date().toISOString().split('T')[0]}_${paymentHeader.payment_type.toUpperCase()}`;

const [newTransaction] = await db.insert(transactions).values({
  name: transactionName,
  invoice_id: paymentLines?.[0]?.invoice_id ?? null,
  account_id: paymentHeader.received_from ?? paymentHeader.paid_to ?? null,
  bank_info_id: paymentHeader.payment_info_id ?? null,
  payment_description: `${paymentHeader.payment_type} - ${paymentHeader.payment_ref} - ${paymentHeader.total_amount}`,
  status: 'Active',
}).returning();
```

### 2단계 — trust_receipt이면 credits 생성

```typescript
if (paymentHeader.payment_type === 'trust_receipt' && newTransaction) {
  const creditName = `TRANSACTION-CREDIT-${new Date().toISOString().replace('T', ' ').split('.')[0]}`;
  await db.insert(credits).values({
    name: creditName,
    credit_amount: paymentHeader.total_amount,
    payment_info_id: paymentHeader.payment_info_id ?? null,
    transaction_id: newTransaction.id,
    status: 'Active',
  });
}
```

### 3단계 — trust_transfer이면 debits 생성

```typescript
if (paymentHeader.payment_type === 'trust_transfer' && newTransaction) {
  const debitName = `TRANSACTION-DEBIT-${new Date().toISOString().replace('T', ' ').split('.')[0]}`;
  await db.insert(debits).values({
    name: debitName,
    debit_amount: paymentHeader.total_amount,
    payment_info_id: paymentHeader.payment_info_id ?? null,
    transaction_id: newTransaction.id,
    status: 'Active',
  });
}
```

### 브릿지 끝 주석 (필수)
```typescript
// === END BRIDGE ===
```

## 안전 처리 — try-catch 래핑 (필수)

위 브릿지 로직 전체를 아래처럼 try-catch로 감쌀 것:

```typescript
try {
  // === BRIDGE: auto-create transaction record ===
  // ... 위 1~3단계 전체 ...
  // === END BRIDGE ===
} catch (bridgeError) {
  // 브릿지 실패해도 payment_header 저장은 성공으로 처리
  console.error('[Bridge] Failed to create transaction record:', bridgeError);
}
```
