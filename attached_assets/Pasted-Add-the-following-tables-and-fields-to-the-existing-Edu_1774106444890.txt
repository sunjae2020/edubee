Add the following tables and fields to the existing Edubee Camp PostgreSQL schema using Drizzle ORM.

1. TABLE: contract_finance_items
   - id (uuid, PK)
   - contractId (uuid, FK → contracts)
   - itemType: 'receivable' | 'payable'
   - itemCategory:
       Receivable → 'client_invoice' | 'agent_invoice'
       Payable    → 'cc_agent' | 'cc_institute' | 'cc_hotel' | 'cc_pickup' | 'cc_tour' | 'cc_settlement' | 'cc_misc'
   - costCenter:
       Revenue → 'RC-CAMP' | 'RC-DIRECT'
       Cost    → 'CC-AGENT' | 'CC-INSTITUTE' | 'CC-HOTEL' | 'CC-PICKUP' | 'CC-TOUR' | 'CC-SETTLEMENT' | 'CC-MISC'
   - label (varchar 255) — display name
   - linkedProductId (uuid, FK → products, nullable)
   - linkedPartnerId (uuid, FK → users, nullable)
   - linkedAgentId (uuid, FK → users, nullable)
   - estimatedAmount (decimal 12,2)
   - actualAmount (decimal 12,2, nullable)
   - currency (varchar, default 'AUD')
   - commissionType: 'percentage' | 'fixed' | null
   - commissionRate (decimal 5,2, nullable) — e.g. 10.00 = 10%
   - commissionFixed (decimal 12,2, nullable)
   - dueDate (date, nullable)
   - paidDate (date, nullable)
   - status: 'pending' | 'invoiced' | 'partial' | 'paid' | 'overdue' | 'cancelled'  (default: 'pending')
   - isAutoGenerated (boolean, default true)
   - isDeleted (boolean, default false)
   - notes (text, nullable)
   - createdAt, updatedAt (timestamp)

2. TABLE: user_ledger
   - id (uuid, PK)
   - contractId (uuid, FK → contracts)
   - financeItemId (uuid, FK → contract_finance_items)
   - userId (uuid, FK → users) — account this entry belongs to
   - entryType: 'credit' | 'debit'
   - costCenter (varchar 30)
   - amount (decimal 12,2)
   - currency (varchar, default 'AUD')
   - description (varchar 500)
   - referenceType: 'invoice' | 'receipt' | 'commission' | 'settlement' | 'manual'
   - referenceId (uuid, nullable)
   - transactionDate (date)
   - status: 'pending' | 'confirmed' | 'reconciled'  (default: 'confirmed')
   - createdBy (uuid, FK → users)
   - createdAt (timestamp)

3. EXTEND existing packages table — add:
   - agentCommissionType: 'percentage' | 'fixed' | null
   - agentCommissionRate (decimal 5,2) — for percentage type
   - agentCommissionFixed (decimal 12,2) — for fixed type

4. EXTEND existing invoices table — add:
   - financeItemId (uuid, FK → contract_finance_items)
   - agentId (uuid, FK → users, nullable)
   - commissionAmount (decimal 12,2, nullable)
   - netAmount (decimal 12,2) — Total Fee − Commission

5. EXTEND existing receipts table — add:
   - financeItemId (uuid, FK → contract_finance_items)
   - confirmedBy (uuid, FK → users)
   - confirmedAt (timestamp)
