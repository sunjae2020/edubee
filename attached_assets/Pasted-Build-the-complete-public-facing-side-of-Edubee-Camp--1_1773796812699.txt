 PART 2: PUBLIC BACKEND APIs
═══════════════════════════════════════════

GET /api/public/packages  (no auth)
  Returns active package groups with:
  {
    id, nameEn/Ko/Ja/Th, descriptionEn/Ko/Ja/Th,
    thumbnailUrl, location, countryCode,
    primaryCurrency, primaryCurrencySymbol, countryFlag,
    interviewRequired,
    spotSummary: null | { grades:[{label,available,total,status}] },
    packages: [{
      id, name, durationDays,
      displayPrice, displayCurrency, displaySymbol, displayFormatted,
      allPrices: { aud, usd, krw, jpy, thb, php, sgd, gbp }
    }]
  }

POST /api/public/applications  (no auth)
  Body: { packageGroupId, packageId, preferredStartDate,
          referralSource, referralAgentCode, primaryLanguage,
          specialRequests, termsAccepted:true,
          participants: [...] }
  Actions:
    - Validate enrollment spot availability per child
    - Auto-generate applicationNumber (APP-2026-XXXX)
    - Create application + participants + grade records atomically
    - Decrement reservedSpots
    - Send confirmation email
  Return: { applicationNumber, message }

 PART 4: APPLICATION MODAL (4-step)
═══════════════════════════════════════════

Max-w-3xl centered modal.
Progress: ① Program → ② Primary Student → ③ Participants → ④ Review

STEP 1: Program Selection
  Program (pre-selected from card or dropdown)
  Package (shows price in program currency: "A$1,800 AUD")
  Preferred Start Date | How heard | Referral Code

STEP 2: Primary Student
  Full Name EN* | Native Name | DOB* (→ "Age: X") | Gender* | Nationality*
  Passport# | Expiry (6+ months validation)
  Grade* — if enrollment spots: "Grade 7 (2 spots left)" / disabled if full
  School | English Level*
  Medical Conditions | Dietary | Special Needs
  Emergency Phone* | WhatsApp | LINE ID

STEP 3: Additional Participants (optional)
  Counter: "👦 1  👨 0  Total: 1"
  Adults (max 4): Name*, Relationship*, DOB, Nationality, Passport,
                  Phone*, Email, WhatsApp, LINE, ☑ Emergency Contact
  Children (max 5): Name*, DOB*, Gender*, Nationality*, Grade* (spot-aware),
                    Passport, Medical, Dietary
  Collapsible cards, [Remove ×] per card

STEP 4: Review & Submit
  Program card + Participants summary + [✏ Edit] links
  ☐ Terms* | ☐ Privacy*
  [Submit Application 🚀]
  Success: "✅ APP-2026-0001 — We'll contact you within 2 business days."

POST /api/public/applications handles submission.
