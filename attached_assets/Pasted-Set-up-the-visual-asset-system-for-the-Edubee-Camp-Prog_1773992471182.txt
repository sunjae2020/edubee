Set up the visual asset system for the Edubee Camp Program Reports module.
This defines logos, illustration symbols, and design tokens used across
all report pages, PDF exports, and the admin UI.

═══════════════════════════════════════════
 PART 1: EDUBEE LOGO COMPONENT
═══════════════════════════════════════════

Create /client/src/components/shared/EdubeeLogo.tsx

The logo is composed purely in SVG — no external image file needed.
This component is used in: Navbar, PDF header, Report cover page,
Login page, and Email templates.

Props:
  variant: 'full' | 'icon' | 'white'
  size:    'sm' | 'md' | 'lg'

SIZE MAP:
  sm  → icon: 24px,  text: 14px
  md  → icon: 32px,  text: 18px  (default)
  lg  → icon: 48px,  text: 24px

LOGO STRUCTURE:
  Left:  Bee icon SVG (see spec below)
  Right: Wordmark "edubee" + "CAMP" subscript

BEE ICON SVG SPEC:
  Viewbox: 0 0 40 40
  Body:    Rounded ellipse, fill #F5821F
  Wings:   Two soft rounded shapes, fill #FEF0E3, stroke #F5821F 1px
  Stripes: 2 curved lines across body, stroke #FFFFFF 1.5px
  Antennae: Two curved paths, stroke #D96A0A 1.5px, dots at tips

WORDMARK:
  "edubee"  → font Inter 700, color #1C1917 (variant='white' → #FFFFFF)
  "CAMP"    → font Inter 500, font-size 60% of edubee, color #F5821F
              letter-spacing: 0.15em, display block below "edubee"

variant='icon' → bee SVG only, no text
variant='white' → all elements white/light for dark backgrounds

Usage examples:
  <EdubeeLogo variant="full" size="md" />   // navbar
  <EdubeeLogo variant="icon" size="sm" />   // collapsed sidebar
  <EdubeeLogo variant="white" size="lg" />  // PDF cover, login hero
