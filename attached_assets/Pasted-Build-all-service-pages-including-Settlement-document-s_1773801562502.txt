═══════════════════════════════════════════
 PART 2: DOCUMENT MANAGEMENT
═══════════════════════════════════════════

DocumentManager component:
  Props: { referenceType, referenceId, mode:'full'|'simple' }
  Embedded in Application [Documents] tab + Contract [Documents] tab

Backend (/api/documents):
  POST   upload (multipart, to Supabase Storage)
         Path: /{refType}-docs/{refId}/{catCode}/{uuid}.{ext}
         Auto-apply defaultDocPermissions | Log 'upload'

  GET    list?referenceType=&referenceId=&group=
         Filter by can_view per role | Never expose filePath
         Include: { permissions: {canView, canDownload, canDelete} }

  GET    /:id/view   → 30-min signed URL | Log 'view'
  GET    /:id/download → 5-min signed URL (attachment) | Log 'download'
  PATCH  /:id/status → approve/reject (CC/AD/SA)
  DELETE /:id → soft delete
  POST   /:id/new-version → version+1, old isLatestVersion=false
  POST   /bulk-download → ZIP via archiver | X-Skipped-Count header
  GET    /checklist?referenceType=&referenceId=
  GET    /:id/access-log (SA/AD only)

Inline Viewer:
  PDF → right slide-over 50%, <iframe src={signedUrl}>
  Image → lightbox modal + zoom
  Other → info card + download
  Expired → red banner + [Upload New Version]

Daily cron:
  Auto-set expired status | Email at 30/14/7 days before expiry
