## Design System Prompt — Edubee Camp Platform

### Brand Identity
Edubee Camp is an educational camp marketplace platform. The design should feel warm, trustworthy, and modern — similar to Claude's UI aesthetic: clean whitespace, soft rounded elements, and a single-accent color palette.

---

### Color Palette (STRICT — do not introduce additional colors)

Primary Accent:    #F5821F  (Edubee Orange — buttons, links, highlights, active states)
Accent Dark:       #D96A0A  (hover/pressed states for orange elements)
Accent Light:      #FEF0E3  (orange tint — tag backgrounds, selected states, subtle highlights)

Neutral 50:        #FAFAF9  (page background)
Neutral 100:       #F4F3F1  (card backgrounds, sidebar)
Neutral 200:       #E8E6E2  (dividers, borders)
Neutral 400:       #A8A29E  (placeholder text, disabled states)
Neutral 600:       #57534E  (secondary text, labels)
Neutral 900:       #1C1917  (primary text, headings)

Success:           #16A34A  (status: active, confirmed)
Warning:           #CA8A04  (status: pending)
Danger:            #DC2626  (status: cancelled, error)

Background:        #FFFFFF  (main content area)
Surface:           #FAFAF9  (app shell, sidebar)

DO NOT use gradients. DO NOT use blue, purple, or green as accent. Orange is the only brand color.

---

### Typography

Font Family: "Inter", system-ui, sans-serif
(Import from Google Fonts: Inter — weights 400, 500, 600, 700)

Scale:
- Display:    32px / 700 / line-height 1.2  — page titles
- Heading 1:  24px / 600 / line-height 1.3  — section headers
- Heading 2:  18px / 600 / line-height 1.4  — card titles, dialog headers
- Body:       14px / 400 / line-height 1.6  — general content
- Small:      12px / 400 / line-height 1.5  — captions, meta info
- Label:      12px / 500 / uppercase / letter-spacing 0.05em — form labels, tags

---

### Spacing System (8pt grid)

4px   — xs  (icon gaps, tight inline spacing)
8px   — sm  (inner padding for chips/tags)
12px  — md  (form field internal padding)
16px  — lg  (card padding, section gaps)
24px  — xl  (between cards, sidebar items)
32px  — 2xl (page section spacing)
48px  — 3xl (major section breaks)

---

### Border Radius

Buttons:    8px
Cards:      12px
Modals:     16px
Inputs:     8px
Chips/Tags: 999px (fully rounded)
Avatars:    50%

### Iconography
Library: Lucide React (consistent stroke-width: 1.5, size: 16px or 20px)
Color: inherit from context (never hardcode icon colors separately)

---

### Motion & Animation
Transition default: all 200ms cubic-bezier(0.4, 0, 0.2, 1)
Page entry: fade-in 300ms + translateY(8px → 0)
Modal open: scale(0.97 → 1) + opacity(0 → 1), 200ms
Skeleton loading: shimmer animation using #F4F3F1 → #E8E6E2 gradient

### Portal Color Coding (subtle left-border accent on sidebar, no new colors)
All portals use the same single-color system.
