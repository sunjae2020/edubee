POST handler — wrap entire logic in db.transaction:

  1. Generate ref: 'PAY-' + Date.now().toString(36).toUpperCase().padStart(8,'0')
  2. INSERT payment_headers
  3. For each line: INSERT payment_lines
  4. Update AR/AP status based on paymentType:
       trust_receipt  → SET ar_status='paid' on contract_products
       trust_transfer → SET ap_status='paid' on contract_products
  5. Auto-generate journal entry based on paymentType:
       trust_receipt  → debit '1200' credit '1300'
       trust_transfer → debit '2100' credit '1200'
       commission     → debit '1100' credit '3100'
       direct         → debit '4700' credit '1100'
     INSERT journal_entries { autoGenerated: true }
